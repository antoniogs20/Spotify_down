#################Descargar musica facil########################
###############################################################
Requerimientos:

1. Tener instalado python a traves de Anaconda (https://www.anaconda.com/)
2. Tener instalado la libreria Spotdl (Inicio+R escribe 'cmd', ejecuta primero 'conda init' 
			     	       cierra y abre el cmd y escribe 'conda activate' 
				       y despues 'pip install spotdl')

Como usar:

1. En la carpeta install hay un archivo llamada 1.bat Haz click.(Se abrira y se cerrara rapido), despues haz click en 2.bat
(crea el entorno de python bien y copia los archivos necesarios)

2.Ejecuta el .exe que esta en el archivo que has descargado, es IMPORTANTE que logo.png este en la misma carpeta que el .exe.

3.Debes introducir la direccion completa de donde este tu archivo listas.txt (Si no lo entiendes dejalo como esta)

4.Debes introducir la direccion completa donde quieres descargar tu musica.

5.Si tienes alguna duda contacta con el que te ha enviado esto.
